Importation de la base de données

Principe général

Vous pouvez charger sur le serveur la base de données permettant d'évaluer les cours. Ces fichiers en .csv doivent respecter rigoureusement les structures rappelées ci-dessous, sans quoi la base de données pourrait être entièrement corrompue.

Il faut noter que la base de données mets quelques minutes à être construite, du fait du nombre important de données évaluées, il faut attendre le message de confirmation avant de continuer les opérations sur le site internet.

Il est nécessaire de vider la base de données à chaque fois que vous chargez les fichiers de cours et d'élèves (une fois au début de chaque semestre).

Fichier élèves

Le fichier d'entrée doit être au format suivant, en .csv avec « ; » comme séparateur. Si les libellés de module contiennent des « ; », le programme ne fonctionnera pas.
Noter que les colonnes et les lignes sont ici inversées pour des questions de visibilité.

Entête de ligne Exemple de ligne
NOM DURAND
PRENOM Jean-Eudes
LOGIN_LDAP jean-eudes.durand@enpc.fr
MAIL jean-eudes.durand@eleves.enpc.fr
DEPARTEMENT GI
LIBELLE_MODULE  Allemand - débutant
CODE_MODULE ALL1
SC_GROUPE 0
PERIODE Hiver
